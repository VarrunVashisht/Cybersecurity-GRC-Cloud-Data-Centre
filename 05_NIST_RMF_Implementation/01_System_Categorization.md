# System Categorization

## 1. System Name
Q-Day Cloud Management Platform (CMP)

## 2. System Description
CMP manages provisioning, monitoring, logging, and access control for customer cloud workloads.

## 3. Information Types
- Customer operational data
- Access logs and audit data
- Configuration and management metadata

## 4. Impact Analysis (CIA)
- Confidentiality: High
- Integrity: High
- Availability: High

## 5. Overall Impact Level
High Impact System
