# Vendor Security Assessment Questionnaire

## Governance
- Do you have an information security policy?
- Is a security leader formally appointed?

## Access Control
- Is MFA enforced for privileged access?
- How is access reviewed and revoked?

## Data Protection
- Is data encrypted at rest and in transit?
- How are encryption keys managed?

## Incident Response
- Do you have an incident response plan?
- What is your breach notification timeline?

## Compliance
- Which security frameworks or certifications do you follow?
