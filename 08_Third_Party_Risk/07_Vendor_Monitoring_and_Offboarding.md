# Vendor Monitoring and Offboarding

## 1. Ongoing Monitoring
- Periodic reassessments based on vendor tier
- Review of security incidents and changes
- Continuous monitoring for high-risk vendors

## 2. Issue Management
Identified issues must have remediation plans and timelines.

## 3. Offboarding
- Revoke access
- Ensure data return or destruction
- Document offboarding completion
