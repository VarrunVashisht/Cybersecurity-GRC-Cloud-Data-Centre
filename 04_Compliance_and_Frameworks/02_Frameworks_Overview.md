# Applicable Cybersecurity Frameworks

## 1. NIST Cybersecurity Framework (CSF)
Primary framework used to structure cybersecurity governance, risk management, and controls.

## 2. ISO/IEC 27001
Used to support Information Security Management System (ISMS) requirements and international customer expectations.

## 3. CIS Critical Security Controls
Provides prescriptive technical controls for cloud security implementation.

## 4. Cloud Security Alliance (CSA) Controls
Addresses cloud-specific security and shared responsibility considerations.

## 5. Regulatory Alignment
Frameworks collectively support regulatory obligations across finance, healthcare, and government sectors.
