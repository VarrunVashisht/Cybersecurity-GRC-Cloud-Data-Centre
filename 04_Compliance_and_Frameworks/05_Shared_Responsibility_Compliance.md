# Shared Responsibility and Compliance

## 1. Cloud Provider Compliance Responsibilities
- Physical security of data centers
- Core cloud infrastructure controls
- Availability and resilience of cloud services

## 2. Customer Compliance Responsibilities
- Secure configuration of cloud resources
- Data protection and classification
- Identity and access management

## 3. GRC Implications
Compliance assessments must clearly identify control ownership to avoid gaps and misunderstandings during audits.
