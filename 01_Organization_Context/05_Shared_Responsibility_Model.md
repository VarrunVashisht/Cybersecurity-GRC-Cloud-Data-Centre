# Shared Responsibility Model

## 1. Cloud Provider Responsibilities
- Physical data center security
- Underlying cloud infrastructure security
- Hypervisor and core platform protection
- Cloud service availability

## 2. Customer Responsibilities
- Data classification and protection
- Identity and access management configuration
- Application-level security
- Secure usage of cloud services

## 3. GRC Implications
Clear understanding of shared responsibility ensures:
- Accurate risk ownership
- Proper control implementation
- Reduced compliance gaps
